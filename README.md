# Лабораторная работа - Игра 2048

## Описание проекта

Данный проект представляет собой веб-реализацию популярной игры 2048 с сохранением прогресса и таблицей рекордов. Игра адаптирована для различных устройств и поддерживает управление как с клавиатуры, так и через сенсорный экран.

## Проделанная работа

### Функциональные возможности

#### Основной игровой процесс:
- Генерация случайных плиток (2 или 4)
- Объединение одинаковых плиток при движении
- Подсчет очков
- Проверка условия окончания игры

#### Дополнительные функции:
- **Сохранение прогресса** - игра автоматически сохраняется в LocalStorage
- **Таблица рекордов** - топ-10 результатов с именами игроков
- **Отмена хода** - возможность вернуться на один ход назад
- **Адаптивный дизайн** - поддержка мобильных устройств
- **Сенсорное управление** - поддержка свайпов на touch-устройствах
- **Клавиатурное управление** - стрелки для перемещения плиток

### Технические особенности

#### Структура проекта:
- `index.html` - основная HTML-структура
- `style.css` - стили и адаптивный дизайн
- `script.js` - игровая логика и взаимодействие

#### Используемые технологии:
- **HTML5** - семантическая разметка
- **CSS3** - Grid layout, Flexbox, CSS переменные
- **Vanilla JavaScript** - читый JS без сторонних библиотек
- **Local Storage** - сохранение данных в браузере

#### Адаптивность:
- Поддержка экранов от 320px до 768px+
- Оптимизированные размеры элементов для мобильных устройств
- Специальные кнопки управления для touch-устройств


## Управление

- **Стрелки клавиатуры** - перемещение плиток
- **Кнопки на экране** - для мобильных устройств
- **Свайпы** - на touch-устройствах
- **Кнопка "Новая игра"** - начать заново
- **Кнопка "Отменить"** - отменить последний ход
- **Кнопка "Рекорды"** - просмотр таблицы лидеров

## Заключение

В результате проделанной работы была создана полностью функциональная версия игры 2048.
